Flex Tools

Set of flexbox based layout class-helpers for HTML tags.

Initializing

f-box enables flexbox. Outer container works as block.

f-align enables flexbox. Outer container works as inline-block.

Aligning

f-align--Y-X controls the align element. Y controls aligning at the vertical axis. X - controls the horisontal one.

Grid

f-grid--N - where N equal of base columns amount

Grid controlled by settings values:

$grid-gutter - gutter between columns

$grid-columns - set column counts

By default we have 10px gutter and 12 column. Enjoy!

Gap

You can control gutters inside flexbox container. You can chose from one of 3 different gaps:

  • Small
  • Middle
  • Large

Gaps worked with grid or any other columns, even outside from flex container.

Pay attention, that gaps realized as margins, so you don't need to blame your code with wrappers, as in padding-based grid systems.

Limitations

Requre flexbox and calc support.

Keep in mind, that current Flexbox implementation has lots of caveats and bugs. Especially in IE.

Perfect for

  • Sketches and prototypes
  • Admin or developer panels
  • Projects on Edge

Don't recommend for:

  • Support for IE 9 and lower
  • If you don't want to handle Flexbox bugs
